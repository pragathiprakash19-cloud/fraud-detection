## Fraud Detection using Data Analysis

## 📌 Overview
This project focuses on detecting fraudulent transactions by analyzing transaction data such as amount and transaction time. It helps identify unusual patterns and distinguish between fraud and normal transactions.

## 🎯 Objective
- Analyze transaction data
- Identify suspicious patterns
- Differentiate between fraud and normal transactions
- Visualize transaction behavior

## 📊 Dataset
The dataset contains:
- Transaction ID
- Transaction Amount
- Transaction Time
- Fraud (0 = Normal, 1 = Fraud)

## ⚙️ Technologies Used
- Python
- Pandas
- Matplotlib

## 📈 Methodology
1. Data Loading and Exploration
2. Checking Missing Values
3. Data Cleaning
4. Converting 0 and 1 into Normal and Fraud
5. Data Visualization (Scatter Plot)
6. Fraud vs Normal Comparison (Bar Chart)

## 📉 Results
- Fraud transactions show unusual patterns in amount and timing
- Visualization helps clearly differentiate fraud and normal transactions
- Data analysis makes fraud detection easier and more understandable

## 📷 Output
- Scatter Plot (Amount vs Transaction Time)
- Bar Chart (Fraud vs Normal Transactions)

## 📌 Conclusion
Fraud transactions often exhibit irregular patterns in transaction amount and time. Data visualization helps in identifying suspicious activities and supports better decision-making.
